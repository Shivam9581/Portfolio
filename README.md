## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Shivam9581/Portfolio.git
```

Navigate to the project directory:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
